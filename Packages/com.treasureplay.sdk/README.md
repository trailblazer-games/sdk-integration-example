# Treasure Play Unity SDK

Treasure Play SDK provides quest delivery, analytics tracking, and deep link handling for partner titles. The SDK ships as a Unity package `.unitypackage`.

## Package Layout

```
sdk/
├─ package.json           # UPM manifest
├─ Runtime/               # Runtime scripts, assets, prefabs
├─ Editor/                # Editor tooling for settings/menus
└─ Samples/TreasurePlayDemo
```
